# customer_behavior_analysis
data analytics project showcasing customer behavior analysis using sql, excel, python, and power Bi.
Data Analytics Project

📌 Overview

This project demonstrates a complete Data Analytics workflow starting from data loading to creating an interactive dashboard and final presentation. The project focuses on understanding the dataset, performing exploratory data analysis (EDA), cleaning the data, analyzing it using SQL, and visualizing insights using Power BI.

The goal of this project is to transform raw data into meaningful insights that can support decision-making.

📂 Dataset

The dataset used in this project contains structured data that is analyzed to identify trends, patterns, and key insights.

Dataset tasks included:

Loading the dataset using Python

Understanding data structure

Identifying missing or inconsistent values

Preparing the dataset for analysis

🛠 Tools & Technologies

The following tools and technologies were used in this project:

Python – Data loading, cleaning, and exploratory data analysis

Pandas & NumPy – Data manipulation and analysis
SQL (PostgreSQL / MySQL / SQL Server) – Querying and analyzing structured data

Power BI – Creating interactive dashboards and reports

NotebookLM – Creating the final PowerPoint presentation

🔎 Project Steps
1. Data Loading (Python)

Imported dataset using Python

Loaded data into a Pandas DataFrame

Checked data types and structure

2. Exploratory Data Analysis (EDA)

Analyzed dataset distribution

Identified trends and patterns

Created visualizations to better understand the data

3. Data Cleaning

Handled missing values

Removed duplicate records

Fixed incorrect data types

Prepared clean data for analysis

4. SQL Analysis

Data was loaded into relational databases and analyzed using SQL.

Databases used:

PostgreSQL

MySQL

SQL Server

SQL tasks included:

Filtering and sorting data

Aggregation (SUM, AVG, COUNT)

Grouping data

Writing analytical queries

5. Power BI Dashboard

An interactive Power BI dashboard was built to visualize key insights.

Dashboard features:

Interactive charts and graphs

Data filters and slicers

Key performance indicators (KPIs)

Trend analysis

📊 Dashboard

The Power BI dashboard presents insights in a clear and interactive format.

Key visualizations include:

Sales / performance trends

Category comparisons

Key metrics and KPIs

Interactive filters for deeper analysis

📈 Results & Insights

Through this project, several insights were discovered from the dataset, such as:

Identification of key trends and patterns

Performance comparison across categories

Data-driven insights for decision making

These insights were summarized in the Power BI dashboard and presentation.
